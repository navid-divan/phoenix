# Parameter Choices

## BFV
- poly_modulus_degree: 4096
- plain_modulus: 1032193
- Security: ~128-bit under RLWE

## ZKP (ABDLOP)
- module_rank d=4, ring_degree=64
- log q ≈ 32 bits
- 128-bit soundness under MSIS/MLWE

## FHS
- n=512, log_q=22, d_max=30
- Verification norm from SampleLeft/SampleRight Gaussian params
